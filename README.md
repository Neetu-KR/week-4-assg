# Financial Sales Data Analysis & Power BI Dashboard

## Project Overview

This project is an end-to-end data analysis case study completed as part of the **Week 4: Data Visualization & Capstone Project**. The objective is to clean, analyze, and visualize a financial sales dataset using Python and Power BI to generate meaningful business insights.

---

## Objectives

- Import and clean the financial sales dataset.
- Perform exploratory data analysis (EDA).
- Handle missing values and duplicate records.
- Analyze sales and profit using Python (Pandas).
- Create an interactive Power BI dashboard.
- Generate business insights and recommendations.

---

- Pandas

- Power BI

## Data Cleaning Steps

The following preprocessing steps were performed:

- Imported the Excel dataset
- Checked dataset dimensions
- Displayed column names
- Verified data types
- Checked missing values
- Removed duplicate records
- Removed extra spaces
- Filled missing values
- Removed invalid or negative values
- Detected outliers
- Saved the cleaned dataset
